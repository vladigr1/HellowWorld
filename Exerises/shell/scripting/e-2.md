# Exercise 2

write bash function `marco` and `polo`.\
* `marco` == current wokring directory is saved
* `polo` == `cd` to directory where you executed `marco`

To use the function you must reload them on the shell.
```bash
source ./marco.sh
```

## marco.sh file

```bash
marco(){
marcoPwd="$(pwd)"
}

polo(){
cd "$marcoPwd"
}
```
